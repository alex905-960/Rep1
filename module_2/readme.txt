ОПИСАНИЕ ПРОЕКТА
*********************************************************************************************
1. Основная цель проекта - получить практику и опыт работы с графическими библиотками
python, использовании python для статистического анализа
2. Задача проекта: выявить факторы, влияющие на результаты госэкзаменов по математике
у студентов
3. Датасет с данными о студентах содержит информацию об их окружении, семье, условиях
обучения
4. Этапы работы: Знакомство с данными, очистка данных, дополнение данных, подготовка
данных к стат.анализу, построение гистограмм отсечение лишних признаков, выполнение
корреляционного анализа, подготовка заключения по результатам анализа
5. Работа выполнялась самостоятельно, были заданы вопросы членам команды, но в итоге
решение принято самостоятельно с учетом мнения команды. Остался доволен результатом
анализа, выяснились как банальные зависимости так и неожиданные. При разработке кода
пришлось сделать пару велосипедов, так как искать способ написания краткого кода 
нехватило времени. Код по стандарту PEP-8 писал по памяти, старался не нарушать основные
правила, но думаю можно было сделать лучше. Из графики использовал только Seaborn. Одну
функцию взял из примера во время обучения, все остальное написано "с нуля". Интересного
узнал про методы мат.статистики, а также графическая библиотка Seaborn понравилась. Главным
результатом считаю, что без постронней помощи смог выполнить работу, результат похож на 
правду. Навыки по работе в pandas и по мат.статистики уже могу использовать в работе.
Планирую изучить дополнительно pandas и seeborn
